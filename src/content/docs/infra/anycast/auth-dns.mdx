## Anycast  Authoritative DNS
Saddle Network's authoritative DNS service is a global anycast DNS network that provides fast and reliable name server services for Saddle Network's domain resources. 

### Underlying Technology
We make use of [CZ.NIC](http://www.nic.cz/en)'s [knot](https://www.knot-dns.cz/) authoritative DNS server. Knot is a high-performance authoritative DNS server that supports modern DNS features and requires minimal resources.
